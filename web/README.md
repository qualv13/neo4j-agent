# DrugPath landing page

A single static page (`index.html` + `assets/`). No build step — host it anywhere:

- **Netlify:** drag the `web/` folder onto [app.netlify.com/drop](https://app.netlify.com/drop).
- **Vercel:** `vercel deploy` from inside `web/`, or import the repo and set the root to `web`.
- **GitHub Pages:** push the repo, then Settings → Pages → deploy from `/web` (or move the files to `/docs`).

When the Aura agent is published, drop its public URL into the hero CTA and the
footer (search for `github.com/qualv13/neo4j-agent` to find the link spots).
